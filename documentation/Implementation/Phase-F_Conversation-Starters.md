# Phase F — Conversation Starters  
**ISOM 360 – Advising Chatbot Project**

## Purpose
Conversation Starters are pre-set example questions that display on the chatbot's home screen. They guide users toward common advising topics.

---

## 1. How to Add Starters
1. Open **Build tab**  
2. Scroll to **Conversation Starters**  
3. Click **Add Starter** for each prompt  
4. Fill in:
   - **Prompt:** what a student might type  
   - **Description:** 1-line summary  

---

## 2. Required Structure
Create **12–18 starters** (2–3 per category).

---
