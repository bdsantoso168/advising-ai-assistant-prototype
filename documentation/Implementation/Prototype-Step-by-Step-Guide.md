# Step-by-Step Guide — Building the Advising Chatbot Prototype  
**ISOM 360 – Advising Chatbot Project**

## Purpose
This guide explains every step from design to implementation to create a functional, no-code advising chatbot using ChatGPT Plus.

---

# Phase A — Prepare Upload Assets
- Review KB micro-chunks  
- Ensure each has ID, title, narrative, source, escalation  
- Store in category folders  

---

# Phase B — Create Your Custom GPT
1. Open ChatGPT Plus  
2. Explore GPTs → **Create a GPT**  
3. Complete:
   - **Name**  
   - **Description**  
4. Click **Save** to enter Build mode  

---

# Phase C — Configure Core Settings
- Web Browsing: **ON**  
- Allowed Sites: official academic pages  
- DALL·E: **OFF**  
- Code Interpreter: **OFF**  
- Memory: **OFF**  

---

# Phase D — Paste Instruction Set
- Copy complete Instruction Set from Design Pack  
- Paste into **Instructions**  
- Confirm all sections appear  

---

# Phase E — Upload Knowledge Base
- Upload each micro-chunk (.docx)  
- Wait for green checkmarks  
- Verify indexing  

---

# Phase F — Conversation Starters
- Add 12–18 example questions  
- Use student-friendly phrasing  
- Confirm each works in Preview  

---

# Phase G — Testing (Sanity Check)
For each category:
- Test 1–2 questions  
- Verify citation  
- Verify escalation  
- Fix KB or instructions if needed  

---

# Phase H — Publish
- Set visibility to **Anyone with link**  
- Save as “Version 1.0 – Prototype Ready”  

---

## Final Implementation Checklist
- [ ] All KB micro-chunks uploaded  
- [ ] Instruction Set pasted  
- [ ] Conversation Starters added  
- [ ] Out-of-scope behavior tested  
- [ ] Preview responses correct  
- [ ] Link documented in project file
